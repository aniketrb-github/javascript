Selectors in jQuery

Dot(.) selector is used to select html elements by their class attribute.
Hash(#) selector is for selecting unique html elements by their ID.

Animation Effects in jQuery

Fade Toggle effect is a combination of Fade In & Fade Out effect.
Similarly, Slide Toggle is a combo of slideUp & slideDown effect.
Whereas the toggle() effect is th combo of show() & hide() effect.

DOM Manipulation using jQuery

We can use following set of methods to manipulate a webpage using jQuery.
- html()
- attr()
- css()
- text()
- append()
- prepend()
- after()
- before()
- remove()
- empty()
- addClass()
- removeClass()
- toggleClass()
